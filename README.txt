To run the code, clone the repo from:

https://github.com/otevet/markov_decision_processes

Install the following required Python 3 packages:
pandas
- Pandas
- Sklearn
- gym
- mdptoolbox-hiive

Run forest-small.ipynb for the small forest management MDP 
Run forest-large.ipynb for the large forest management MDP 
Run frozen-small.ipynb for the large frozen lake MDP 
Run frozen-large.ipynb for the small frozen lake MDP
